练习，落地！

申论

发现问题，分析问题，解决问题
是什么，为什么，怎么办！

客观题

概括题

如何识别题型
提问特点：准确、全面、简明
准确：概括过程不能曲解原文意思，如扩大或缩小范围。
全面：踩点给分
简明：简单明了，不要啰嗦，减少不必要的关联词。

书写要求

  1. 短语、短句、规范表达、多样表达大（不要口语）
  2. 总括句、分类、分条

阅读技巧
高频词汇：意识、监督、管理、宣传、投入、设施、人才···
问题描述：不足、不力、淡薄、难以、无法、缺失···
人物对话：权威、普通人
权威：专家、学者、教授··· 一般都是对的
普通人：市民、网民、张某··· 放映问题

单一概括

提问特点：谈谈**的具体表现；概括一下材料主要问题、原因、对策（其中一方面）

难点：

  1. 分层（分类）

    并列
    表述：主体+问题
    主体：不等于主语，包含主语；可以是人、物、设施、意识······
    高频词:表达，材料

    1. 常规分层：
      一. 主体分层
      1. 政府、企业、民众、家庭
      2. 景点、地名

二. 多角度

  1. 意识观念问题：1 道德观念2 法律意识3 政绩观
  2. 法律制度监督问题：1长效机制 2 法律制度不完善3 强制力不足4 违法成本低
  3. 公众监督： 电视、新闻、媒体曝光举办力度不足；网络监督缺位
  4. 教育宣传：1 转变观念 2 培训技能 3 营造氛围 4 宣传力度不足
  5. 投入：资金、经费、政策、技术、人才、设施不完善
  6. 监管（政府）：处罚不严、多头管理、缺乏协作、权责不明、监管不力

三. 特殊分层：

  1. 内因、外因
  2. 主观原因、客观原因
  3. 自身原因、外在原因
  1. 规范表达
  2. 观点全

阐述概括

提问点：对某名词下定义或者解释。

答题：问题 是什么；原因 为什么； 对策 怎么办。

答题注意分类：
问题：1、······ 2、······
建议：1、······ 2、······

  1. 解释词汇
    解题思路：字面含义+引申含义+个人评价
    答案组织：······是指；表现在······；应该······
  2. 解释句子
    拆分成词去解释
    解题思路：引申含义+个人评价
    答案组织：表现在······；应该······

综合概括

对策题

提问特点：对问题提出对策，解决办法等。有针对性、切实可行、内容具体

写对策：抽象对策+具体对策
强调怎么做，做什么。

一、意识

  1. 科学规划……，合理布局……
  2. 树立科学长远观念
  3. 树立……的意识
  4. 政府重视
  5. 建立……绩效考核机制

二、法律制度

  1. 完善……法律制度
  2. 建立……长效机制
  3. 提高违法成本，加大惩罚力度
  4. 做好顶层设计

三、监督

  1. 引发公众关注
  2. 引发公众舆论监督
  3. 提高社会参与度
  4. 加大曝光、举报力度

四、宣传教育

  1. 引导……转变观念
  2. 提升……职业道德、自律意识
  3. 培训……技能
  4. 营造……氛围
  5. 加强宣传教育

五、监管

  1. 加大处罚力度
  2. 成立专门机构
  3. 加强部门的协作配合

六、投入

  1. 拓宽融资渠道、增加资金收入
  2. 政策支持
  3. 增加人才、技术、设备投入
  4. 完善技术标准、提高技术门槛
  5. 完善硬件设施

公文写作

分析题

提问特点：分析合理、观点准确；谈谈对**的理解；

大作文

提问特点：宣传稿 、一封信、倡议书、提纲、纲要、内容要点······
