2019-01-19-战争模拟-长夜如箭.md

发表于 2019-01-19 | 分类于 科幻
字数统计: 2,156 | 阅读时长 ≈ 7

  西突帝国的先进力量反扑失败后。帝国开始大清洗，甚至有人喊出恢复死刑，要处死背叛帝国的军人。全国近一半的公职人员，老师教授，军队人员因支持先进力量被迫离职离岗。埃苏丹的权利得到空前的巩固，帝国复辟梦也越发强烈。大有突厥振兴的味道。埃苏丹的政策引起盟友和米帝的警惕，接二连三警告埃苏丹要保持克制，与北极熊保持距离。然而为时已晚，埃苏丹向北极熊纳投名状，自断财路，关闭边境线。叙政府军在北极熊那火树银花般空中力量的配合之下，竟有破竹之势，收复几年苦战未有战果的城市。

宁为鸡头，不为牛尾-再微小的阴谋家也不会甘心做别人的傀儡。叙政府正品尝到胜利的滋味时，埃苏丹宣布为了打击库尔德恐怖分子，西突帝国派出精锐的陆军部队入侵叙利亚。全世界都为之惊谔。有人理论纷纷：埃苏丹在平衡米熊关系。也有人支持：埃苏丹打击分裂势力。更多的人担心：第三次世界战争就此拉开序幕。兵强马壮用来形容这个欧洲第二强陆军一点都不为过。大家预估库尔德女兵们的末日来了，西突可能永久占有叙北部山区。一个月过后，意外的是欧洲第二强陆军的钢铁洪流既然举步维艰，仅在边境地区的开阔地区停留。外界猜测西突战力如此不堪是否与半年前的大清洗有关。有能力的军人被革职，士兵士气低落。而西突军人也想不透，库尔德的女兵为何如此强悍，如有神助。
　　夕阳西下，残云血红。西突士兵快速换岗，开启刚刚从腐国进口的反火炮雷达，以防止库尔德人的偷袭。几个小时前，一只小型的先遣重装甲车队向山区突击，没想到被库尔德妹子用rpg偷袭车队。这支重装甲车队全被歼灭。这也是一个月军事行动来第十九支被歼灭的装甲车队了。而且大部分都是白天被偷袭。为此西突的空军把山区都炸矮了几米。除了石头泥土，看不到任何或者的行走动物。而每每西突对地面大轰炸后，装甲部队向山区推进时都被不明火力偷袭。西突士兵死死地盯着雷达作业台，虽然有自动报警系统提醒是否有异常，但是士兵还是高度紧张着甚至不那么信任新装备的雷达。天空中星星渐渐多了，除了星光，士兵不愿意看到多余的东西。也许他们更多的是后悔来这人间炼狱。
　　东方泛白，草露结霜。西突夜里已经换防了两次，幸运的是，昨夜安然度过。西突国内媒体如复读机般宣传，帝国军队又取得那些战果。而西方媒体则老不留情面的揭了盟友的丑，把昨天白天那起失败的突袭的画面不断的重播，似乎再警告西突人，还是要乖乖听米帝的安排，不要强出头，否则迟早和那主力坦克的炮塔一样四分五裂。埃苏丹知道米帝不开心，但是为了转移国内矛盾，不得不出如此下策，是否是下策还不得而知呢。埃苏丹在电视直播面前故作镇定，警告米帝不要给库尔德恐怖分子提供武器。他有消灭一切分裂势力的能力。由于埃苏丹曾得罪过北极熊，现在得罪米帝，到底谁给库尔德女兵武器，他也一头雾水。难道是犹太人？他被困扰着。
　　星河灿烂，西瓜电扇。石狗在为《战争模拟》中角色的战力值重做体能测试。游戏中的角色体能通过一定的算法模拟玩家的真是体能，简单说光充币也得不到高战斗力，必须是真实体能加上rmb才能提高角色体能。这个算法正好是致远单车室内训练台上得来的。车手可以在致远训练台上与世界各地车友联网比赛，但由于时差关系，某些车手就录制自己骑行的数据，通过虚拟人重复该数据与其他车手同场竞技。而《战争模拟》防止堕落成为肥宅游戏，玩家可以自行定运动计划，并通过提高运动成绩，进而提高游戏中角色体能。游戏是致远的，身体是您的。作假提高数据一点意义都没有，致远已经限缩鼠标版的ID（保留对残疾人特殊照顾）全力推广运动版游戏（身体运动直接投射到游戏中）。
　　“铃…”石狗正在模拟器上做最后的冲刺，无暇顾及旁边的手机。
　　“喂，您好！石总正忙，一会他会回你电话。”石狗的秘书接了电话。
　　“不必了，你告诉他，他设计的电磁滑翔机非常棒。”电话直接挂了。
　　“哇！”秘书吃惊，这语气赶上女友教训男友。
　　石狗看着模拟器上的结果，上气不接下气，这成绩提高的越来越有难度。难道是最近通宵多了，体能下降！
　　”石总，”
　　“我听到了”石狗打断秘书的话，像刚刚被女友训话生闷气的男生。“我设计的东西，当然好了。可惜！”
　　“可惜只能在游戏中嘛！”关平不知道从哪里冒出来。“夜宵？”关平说。
　　“好！”
　　“没你事。”石狗否定秘书的回答“都不知道，关总葫芦买什么药。你敢吃他夜宵？嫩！”
　　“补药，就天热，请你们吃小龙虾。”关平说。
　　“粤西土著对小龙虾不感冒。”秘书失望的说。
　　“刚刚，赵忻来电话了，说了莫名其妙的话，就一句话。就挂断了。”石狗又说到电话。
　　“偷袭利器嘛！来无影，去无踪！你说，赵忻会不会是用什么账号？”关平说。
　　“别动运营服务器的主意，她是什么账号都不要紧。人家有天网，把你一举一动都看的透透的。”石狗恢复了一下气息。
　　“夜宵吃什么？”秘书还是没有放弃。
　　“烧耗！”石狗和关平异口同声。
　　艳阳高照，寸草不生。个别西突的士兵因为压力过大，在军营里正准备吞枪自杀。“轰！”一声巨响，冲击波震倒士兵，枪管抠下门牙。士兵还来不及喊疼。“轰，轰！”营地的军火库被偷袭了。军营里的士兵如热锅上的蚂蚁到处乱串，且无路可逃。刚刚进口的雷达也被反辐射导弹击中。西突这个前沿营地彻底瞎了。而在西突领空上盘旋的预警机也似乎收到营地发送的求救信号，马上指挥境内的战斗机起飞出境，提供空中支援，地面的周围营地马上提高戒备。埃苏丹一直对北极熊控制叙北部山区的领口表示不满，希望通过控制打击库尔德女兵，而不是地面部队。而北极熊表面表示理解，实际s-400几次瞄准南下的西突飞机。埃苏丹也是有口难言，毕竟人家是世界第二军事强国啊。这次军方也压制不住自己小火山了，已经是第二十次被偷袭。而偷袭的人员既然毫发无损，自己且死亡惨重。不到半小时，西突多架战机已经组对飞抵叙北领空。然而，除了火烧的战营，什么都没有，不但库尔德女兵的影子看不到，她们可能使用的交通工具也没有。难不成她们都长了翅膀？

奸佞当道，世如长夜。
　　忠孝难为，暗箭破晓！
