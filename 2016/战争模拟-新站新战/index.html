战争模拟-新站新战

发表于 2016-07-28 | 分类于 科幻
字数统计: 1,634 | 阅读时长 ≈ 5

  石狗送走赵忻，回家洗漱一番就赶回公司与关平商量与赵忻所服务的研究所（中国人民解放军蓝军未来战争模拟研究所）合作的事情。“关平十有八九是支持的，于公于私与政府部门合作是十分有利于公司的战略发展。即使这样的深度合作会影响到公司的海外市场的开拓。”石狗想。
石狗匆匆赶到致远，而关平正在会议室与前潮资本的代表在洽谈融资。石狗不便前去打断或加入，自《战争模拟》立项后，他就把所有精力放入游戏策划，而把致远的管理全数交给好搭档关平。石狗坐到自己的设计的座躺工作椅（结合人体曲线设计舒适的座椅，也可以展开成卧躺模式，方便午休等）与自己对话（自我审问），确定自己是否被私人情感左右了判断。赵忻需要《战争模拟》的文案作为她研究未来战争的参考资料，同时有价值的文案将被过滤不能公开，这样的话，《战争模拟》的策划文案的同事都要签保密协议，说不定还要被监控起来，也许赵忻早就监控他了。但是如果《战争模拟》能给中国军队的建设提供力所能及的帮助也是实现了他的军事梦。
“滴！您有新邮件！”石狗的手表提醒他。又是谷子的新邮件，除了例行问候，还特别提到叙利亚内战平定后欧洲各国强力遣送难民问题。

这两年与谷子交流的深入，石狗对她的感情日趋复杂，是钦佩、怜悯、厌恶、喜欢。石狗都找不到准确的形容词来形容这个战地名记。有空时候或者《战争模拟》策划没有灵感的时候他会经常翻翻与谷子的邮件，从中汲取力量和灵感。
邮件一（石狗发）：
“尊敬的谷子女士：
您好，我叫石狗，小鑫的一位朋友。在信鸽联盟上一直关注你的动态和报道。听小鑫说，他在比雷埃夫斯港遇到你，听说你受伤了，不知现在伤情如何？如果需要我的帮助，请给予告知，我当尽力而为！
顺颂夏安
友：石狗
2018.06.01”
邮件二（谷子发）：
“石先生，谢！无忧，勿念！”虽然寥寥几个字，还是让石狗高兴几天，感谢小鑫的牵头，没让他的邮件在这位赫赫有名的战地名记得邮箱里石沉大海。不过接下来的几封邮件都没有收到谷子的回信。也许她很忙吧，石狗安慰自己。
邮件三（谷子发）：
“尊敬石狗先生
您好，非常抱歉没有回复你的邮件。上次回复你的时候刚刚完成一个小手术，用电脑不是很方便，所以惜字如金内容简短，请勿见怪！、、、、、、”
石狗再次收到谷子的回信已是上次回信的一个月后。谷子在信中提到自己如何受伤，如何被转运。并抄送了她最新的新闻稿，关于叙利亚内战和难民问题。其中一句话深深触动了石狗，“战争不过是欲望的延长，权力者的游戏。”“战争只是游戏？这是对生命的何等蔑视，对正义的何等、、、”
“石狗”关平走到跟前把石狗从回忆中拉了出来。
“客户呢？”石狗问，不过情绪中没有以往一样追问会谈的结果。这次融资对第三代《战争模拟》和后续反正非常重要。几乎是决定了《战争模拟》的下一步走向。
“这次谈的不错，还有些细节问题，前潮的代表还要回去请示他们的领导。”关平察觉得到这些细节。
“嗯，我有个好消息告诉大家，大家开个小会吧！”石狗说。
大家陆续来到会议室，关平突然想到早上赵忻来过的事，正要给石狗说。
“这里有个重大喜讯要给大家宣布一下。”石狗略有激动的说“今后我们融资再也不是问题了，今天早上我找到了可靠的战略合作伙伴了。”
“哗”大伙高兴鼓掌
“我们的第二代《战争模拟》非常成功，引起了很多军事专家的关注，甚至也打动了中国人民解放军，今天早上我收到了中国人民解放军蓝军未来战争模拟研究所主任的口头要求，希望和我们合作。”石狗解释。“但毕竟致远是大家的，这个重要的决定需要大家一起决定，大家都说说你们的意见。”石狗向来都是先民主后独裁，听取意见民主，决策和执行独裁。
“我觉得先评估一下为好，毕竟对方是军队机构啊。”关平的担忧，让大家有些不理解，毕竟自己的军队找上门合作是多美的事，又不是鬼子的皇军。
“评估一下也好，《战争模拟》有了中国军队背景是否对我们的海外市场开拓添加不必要阻力。当然要比较弊大利大。”丁一说。
“还评什么，绝对好机会啊，多大的金主啊！”一个同事反驳。
“、、、、、、”
“先吃饭！”饭点闹钟准时响了。
“我知道你已经决定和赵忻合作，当还是要比较一下利弊。无论你决定是什么我都会支持你。”吃饭时候关平对石狗说“当然你也要避免感情用事，不要忘记你们敢分手时候要死要活的样子啊。”关平略带嘲讽。
“滚”石狗瞪他一眼，在致远只有少数几个元老敢拿石狗开刷“我知道该怎么做。”
饭后石狗又留下加班，给赵忻回复邮件，让她将合作文件发致远的律师顾问。之后又给谷子回了邮件，交流叙利亚难民遣返问题、、、、、、

ps：解决难民问题最佳方案是结束战争。
